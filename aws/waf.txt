NAME

DESCRIPTION
       NOTE:
          This is AWS WAF Classic documentation. For more information, see AWS
          WAF Classic in the developer guide.
              For the latest version of AWS WAF , use the AWS  WAFV2  API  and
              see  the  AWS WAF Developer Guide . With the latest version, AWS
              WAF has a single set of endpoints for regional and global use.

       This is the AWS WAF Classic API Reference for  using  AWS  WAF  Classic
       with  Amazon  CloudFront.  The  AWS  WAF Classic actions and data types
       listed in the reference are available for protecting Amazon  CloudFront
       distributions.  You  can  use these actions and data types via the end-
       point waf.amazonaws.com . This guide is for  developers  who  need  de-
       tailed  information  about the AWS WAF Classic API actions, data types,
       and errors. For detailed information about AWS WAF Classic features and
       an  overview  of  how  to  use the AWS WAF Classic API, see the AWS WAF
       Classic in the developer guide.
