NAME
       untag-attendee -

DESCRIPTION

       See also: AWS API Documentation

       See 'aws help' for descriptions of global parameters.

SYNOPSIS

          --meeting-id <value>
          --attendee-id <value>
          --tag-keys <value>
          [--cli-input-json <value>]
          [--generate-cli-skeleton <value>]

OPTIONS
       --meeting-id (string)
          The Amazon Chime SDK meeting ID.

       --attendee-id (string)
          The Amazon Chime SDK attendee ID.

       --tag-keys (list)
          The tag keys.

          (string)
