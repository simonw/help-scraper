NAME
       cli-dev -

DESCRIPTION
       Internal  commands for AWS CLI development.  These commands are not in-
       tended for normal end usage. These commands are used  to  help  develop
       and  debug  the AWS CLI.  Do not rely on these commands, backwards com-
       patibility is  not  guaranteed.   Any  of  these  commands  under  "aws
       cli-dev" may be removed in future versions.

       See 'aws help' for descriptions of global parameters.

SYNOPSIS
          aws cli-dev [dev-command]

OPTIONS
       None

       See 'aws help' for descriptions of global parameters.
