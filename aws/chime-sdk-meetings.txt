NAME
       chime-sdk-meetings -

DESCRIPTION
       The  Amazon  Chime SDK meetings APIs in this section allow software de-
       velopers to create Amazon Chime SDK meetings, set the AWS  Regions  for
       meetings, create and manage users, and send and receive meeting notifi-
       cations. For more information about the meeting APIs, see Amazon  Chime
       SDK meetings .
