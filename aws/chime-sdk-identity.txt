NAME
       chime-sdk-identity -

DESCRIPTION
       The  Amazon  Chime SDK Identity APIs in this section allow software de-
       velopers to create and manage unique instances of their  messaging  ap-
       plications.  These  APIs provide the overarching framework for creating
       and sending messages. For more information about the identity APIs, re-
       fer to Amazon Chime SDK identity .
