NAME
       cognito-idp -

DESCRIPTION
       Using  the Amazon Cognito user pools API, you can create a user pool to
       manage directories and users. You can authenticate a user to obtain to-
       kens related to user identity and access policies.

       This API reference provides information about user pools in Amazon Cog-
       nito user pools.

       For more information, see the Amazon Cognito Documentation .
