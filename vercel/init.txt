
  ▲ vercel init [example] [dir] [-f | --force]

  Options:

    -h, --help        Output usage information
    -d, --debug       Debug mode [off]
    -f, --force       Overwrite destination directory if exists [off]

  Examples:

  –  Choose from all available examples

      $ vercel init

  –  Initialize example project into a new directory

      $ vercel init <example>

  –  Initialize example project into specified directory

      $ vercel init <example> <dir>

  –  Initialize example project without checking

      $ vercel init <example> --force
